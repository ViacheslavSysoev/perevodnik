# Доменные правила перевода

Этот файл — место для **специфичных для конкретной книги** правил оформления, которых нет в общем `TRANSLATION_GUIDE.md`. Сюда дописывай:

- доменную терминологию и её особенности (например, традицию транслитерации имён);
- нотацию формул (mhchem для химии, AMS-пакеты для математики, обозначения физики);
- особые admonition-классы, которых нет в шаблоне;
- любые отклонения от ГОСТ 8.417 (старая система СГС, инженерные единицы и т.п.);
- предпочтения по терминологии, которых нет в `GLOSSARY.md` ещё (например, выбор школы перевода).

`TRANSLATION_GUIDE.md` универсален. `DOMAIN.md` — конкретен под эту книгу.

Агенты-переводчики читают **сначала `DOMAIN.md`**, потом `TRANSLATION_GUIDE.md`, потом `GLOSSARY.md`.
