# Прогресс перевода

`[ ]` — не начато (pending), `[~]` — в работе (draft), `[r]` — на ревью (review), `[x]` — готово (done).

Жизненный цикл раздела: переводчик `[ ] → [~]`, ревьюер `[~] → [r]`, сборщик `[r] → [x]` после успешного `mkdocs build --strict`.

**Не редактируй этот файл вручную при инициализации книги.** Его автоматически заполняет `core/scripts/render_mkdocs.py` (или эквивалентный шаг в `init_book.py`) из `chapter_map.json`. Дальше его обновляют агенты по ходу работы.
