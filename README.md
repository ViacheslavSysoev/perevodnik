# perevodnik

**Open-source инфраструктура автоматического перевода книг через Claude Code.**

PDF на входе → MkDocs-сайт на выходе. Бесплатно, на твоей подписке. С формулами, единой терминологией, иллюстрациями, кнопкой «Редактировать» для правок через PR.

> «Знание, к которому есть лицензия, но нет языка — это знание, к которому нет доступа.»

## Идея

Знание лучших университетов мира лежит под открытыми лицензиями — но между ним и читателем стоит языковой барьер. Платные сервисы автоперевода технических книг работают плохо: терминология плывёт, формулы рассыпаются, ссылки на рисунки уезжают в никуда.

`perevodnik` решает это иначе:
- **Декомпозиция вместо одной большой сессии.** Один Claude-агент = один раздел. Координация через файлы (`PROGRESS.md`, `GLOSSARY.md`).
- **Глоссарий растёт сам.** Каждый агент сверяется со словарём, новые термины фиксирует там же. Следующий агент уже видит решение.
- **Сканы — источник истины.** `pdftotext` теряет индексы и формулы; агент сверяется с PNG-страницами.
- **Возобновляемость через файловую систему.** Лимиты кончились — закрыл терминал. Через сутки сказал «продолжай» — пайплайн поднялся с того же места.

## Быстрый старт

**Требования:**
- Python 3.10+
- [Claude Code](https://docs.claude.com/en/docs/claude-code/) (CLI)
- [poppler-utils](https://poppler.freedesktop.org/) (`pdftotext`, `pdftoppm`, `pdfimages`)

**Установка:**

```bash
git clone https://github.com/ViacheslavSysoev/perevodnik.git
cd perevodnik
pip install -r requirements.txt
```

**Перевод книги:**

```bash
cp ~/my-book.pdf books/
claude   # запускаем Claude Code в корне perevodnik/
```

В Claude:

```
переведи books/my-book.pdf на русский
```

Claude задаст пару уточняющих вопросов (лицензия, атрибуция, опционально URL репозитория), затем запустит экстракцию, картографа, переводчиков параллельно. Когда лимиты кончатся — закрой Claude. Через час/день/неделю:

```
claude
> продолжай
```

Пайплайн подхватит с того же раздела.

**Локальный предпросмотр:**

```bash
cd result/my-book
mkdocs serve   # → http://127.0.0.1:8000
```

## Что лежит внутри

```
perevodnik/
├── CLAUDE.md                 # инструкции для Claude (читается автоматически)
├── core/                     # каркас, общий для всех книг
│   ├── templates/<lang>/     # i18n-шаблоны (TRANSLATION_GUIDE, GLOSSARY, mkdocs.yml.j2 …)
│   ├── scripts/              # extract.py, init_book.py, progress.py, render_mkdocs.py
│   └── .claude/              # commands и skills для Claude Code
├── books/                    # ВХОД: только PDF, ничего больше
├── result/                   # ВЫХОД: всё, что генерирует пайплайн
└── examples/                 # ссылки на референс-имплементации
```

## Поддерживаемые целевые языки

В шаблонах сейчас лежит `ru/` и заготовка `en/`. Добавить новый язык — скопировать `core/templates/ru/` в `core/templates/<lang>/` и перевести `TRANSLATION_GUIDE.md` (всё остальное — данные/конфиги, не зависят от языка).

## Ограничения

- PDF должен иметь текстовый слой. Сканы без OCR — сначала через `ocrmypdf`.
- RTL-языки (арабский, иврит) и языки без MkDocs Material search — пока не тестировались.

## Вклад
